AI图像处理系统

项目概述

该项目是一个基于TensorFlow框架的智能图像处理系统，主要实现了图像分类、特征提取和场景识别功能。 系统能够对输入的图片进行分析，自动识别其中的物体类型、场景特征等信息，并进行分类整理。

技术架构

后端：TensorFlow + Python + Flask
前端：Vue.js + Element UI
数据存储：MySQL + Redis
服务器：Docker + Kubernetes

我的职责

- 负责TensorFlow模型的训练与优化，实现85%的图像分类准确率
- 开发前端交互界面，实现图像上传、结果展示与数据管理功能
- 优化模型推理速度，实现批量图像处理流水线，处理效率提升40%

健康管理系统

项目概述

健康管理系统是一个全栈开发的健康数据管理平台，旨在帮助用户记录和管理个人健康数据， 包括体重、心率、血压等健康指标，并提供数据可视化与健康建议功能。

技术架构

后端：Spring Boot + Spring Cloud + MyBatis
前端：Vue.js + Vuex + Element UI
移动端：Android SDK + Kotlin
数据库：MySQL + Redis

我的职责

- 负责用户管理模块的前后端开发，实现用户注册、登录与权限管理功能
- 开发健康数据统计与可视化模块，使用ECharts实现多维度数据展示
- 优化数据库查询性能，实现健康数据的批量导入与导出功能
